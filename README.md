# Clipboard Manager

إدارة الملاحظات والنصوص المحفوظة مع إمكانية التوسيع المتعدد والمقارنة بسهولة.

## الميزات

- إنشاء وتعديل الملاحظات
- تنظيم الملاحظات في تبويبات مختلفة
- توسيع عدة ملاحظات في نفس الوقت للمقارنة
- البحث في الملاحظات
- ترتيب الملاحظات حسب التاريخ أو المحتوى
- تصدير واستيراد البيانات
- دعم اللغة العربية مع اتجاه النص المناسب
- واجهة مستخدم بسيطة ومتجاوبة

## كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. أضف ملاحظات جديدة باستخدام حقل الإدخال
3. انقر على الملاحظة لتوسيعها أو طيها
4. استخدم القائمة المنسدلة لكل ملاحظة للتعديل أو النقل أو الترجمة
5. استخدم الشريط الجانبي لإدارة التبويبات

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage للحفظ المحلي

## التثبيت

لا يتطلب التثبيت، فقط افتح `index.html` في أي متصفح حديث.
